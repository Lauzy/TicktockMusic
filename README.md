# TicktockMusic

## 介绍
「Ticktock」一款 material design 风格的音乐播放器，可以播放本地和网络音乐，并且提供歌词显示。项目采用 clean architecture、mvp、rxJava2、retrofit2、dagger2 进行开发，代码结构清晰，层次分明。

## 特点
- Material design
- MediaSession + MediaController 控制音乐播放
- 本地音乐 + 网络歌曲
- 播放队列
- 文件夹、我的喜欢、最近播放等
- 支持歌词显示及同步
- 耳机线控、通话及多音频播放时暂停
- 动态换肤
